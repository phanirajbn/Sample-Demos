Canvas, a new Element in HTML5

Canvas is an element inside HTML5 to create 2D and 3D graphics Images. Unlike Images, they are created at runtime with some JS logic and at design all U see is a white element. Typically Canvas is like White board on which programmers draw the content using Scripting languages. The logic of developing the Image is dependent on the requirements of the client. There are some 3rd party tools that can be used for developing these 2D and 3D images.

How it works?

For working with Canvas object, U need a drawing meterial called Context Object. The Context Object is the one which contains the drawing elements required to draw on the white board(Canvas). U first need to create this object which is determined whether U want to create a 2D Graphic or a 3D Graphic Image. The Context object is used to define how U wish to draw the image on the Canvas.
Later we use the Javascript APIs that refer to many GDI objects like Pens, Brushes, bitmaps and soforth to draw on the Canvas thro the Context Object.
