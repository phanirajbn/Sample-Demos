Drag Drop Features of HTML5

HTML allows the dragging and droping of the content from one part of the page to another, The element that is supposed to be dragged must be set with the attribute called Draggable to true. This Feature will help to create Web Portal Apps. The element on which the draggable object is droped must be set with an attribute called allowDrop to true. U R going to use Javascript to handle the Drag drop events of the DOM elements.

The important events are OnDrop and onDragOver. The Dragging object must handle an event called onDragStart. The onDrop is triggered when the object is droped into the droppable element, the onDragOver is invoked while the object is moved on the dropping element.
