About Audio and Video Streaming...

Unlike old HTML Pages, U need not depend on Third party controlls to display Video and play Audio files. Earlier these things were to be displayed using components like Flash Player and Silverlight. Now HTML5 offers <audio> and <video> tags for streaming the Media files.

New in HTML5 is the feature of new Tags to facilitate the Document Programmers to define the Tags based on the intention of placing them in the document. Typically a document may have one or more articles and each article is supported by headers, Sections and Footers. Earlier these things were done using Div elements with different class names defined thro CSS files. Elements would provide the information on the intention of the document.
