What is JSON?

JSON is a simple way of presenting data to the User Interface Applications as Javascript objects. Its a std defined for xchanging the data between different Platforms. Unlike XML you dont need parsers or provide any kind of Schemas to view them. They simply represent the data in form of var. This is called Javascript Object Notation.

JSON is a pure Text based data Presentation. It does not need any high end protocols for data Transfer. Its used very much for transfering data from the Web Services which are exposed as RESTFull Services. Representational State Transfer is a mechanism of sharing the Service data thro simplier formats like JSON and XML.
