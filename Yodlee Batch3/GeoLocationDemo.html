Using Geolocation Features of HTML5

Geolocation APIs are new features introduced in HTML5 to use the GOOGLE APIs to get the latitude and the longitude of the the user and display the info appropriately. But for the success of this, there are various Factors

  1. U must be connected to Internet
  2. The User should provide permissions for the Google APIs to locate U.
  3. Accuracy is based on various factors liek GPS, WIFI, EDGE, Connectivity options....
  4. We use GOOGLE APIs to get the info about the Maps and display them as static or dynamic Content
